# Функциональность приложения

Приложение — мобильный/веб-портал Национальной библиотеки Беларуси (НЛБ) в формате PWA.

## Главная страница (`/`)
- Hero-секция, преимущества, услуги, бестселлеры, события, новости, проекты, отзывы, партнёры, контакты, FAQ

## Услуги
- `/services` — каталог услуг
- `/services/:id` — детальная страница услуги
- Отдельные витрины: `/services/cafe`, `/services/kids-zone`, `/services/lecture-hall`, `/services/concert-hall`

## Полиграфические услуги
- `/printing-services` — каталог
- `/printing-services/:id` — карточка услуги
- `/printing-services/order-confirmation` — подтверждение заказа

## События
- `/all-events` — список всех событий с фильтрами
- `/all-events/:id` — детальная страница события с регистрацией

## Новости
- `/all-news` — список с **поиском по заголовку/описанию/тегам** и **фильтрами по категориям**
- `/news/:id` — детальная страница (hero, sticky-сайдбар, шаринг через `navigator.share`, блок «Похожие новости»)
- 15 новостей с расширенной структурой: `slug`, `category`, `author`, `tags`, `fullContent`

## Книги
- `/books-catalog` — каталог
- `/books/:id` — детальная страница книги

## Карты
- `/interactive-map` — интерактивная карта здания библиотеки (кликабельные зоны)
- `/minsk-map` — карта услуг по Минску

## Профиль и аутентификация
- `/registration` — регистрация
- `/profile` — личный кабинет
- `/notifications` — центр уведомлений

## PWA-возможности
- Установка приложения на главный экран (Android/iOS/Desktop)
- Нижний баннер с предложением установки и cooldown 7 дней через `localStorage`
- Оффлайн-режим через Service Worker (Workbox)
- Манифест: standalone, иконки 192/512, theme-цвета
- Push-уведомления (баннер запроса разрешений)
- Автоматическое отключение SW в Lovable-превью

## UX-фичи
- FAB-виджет (плавающая кнопка действий)
- Форма обратной связи (`FeedbackForm`)
- Адаптивная вёрстка (mobile-first)
- Анимации появления (Motion / tw-animate-css)
- Тосты-уведомления (Sonner)

## Источники данных
Локальные TypeScript-модули в `src/app/data/`:
- `news.ts` — новости
- `events.ts` — события
- `printing-services.ts` — полиграфия
- `service-details.ts` — описания услуг
